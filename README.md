# Zeenode Self-bot

## How to use?

    Launch "install.bat" to install all modules.
    
    Put in main.py your prefix for commands. (Line 6.)
    
    Put in config.json your token.
    
    Launch "launch.bat" to launch Zeenode SelfBot.
